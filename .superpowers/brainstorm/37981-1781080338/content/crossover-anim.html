Cross-Over Animation Pipeline

4 sequential Framer Motion frames. Each frame completes before the next fires via onAnimationComplete.

Frame ① — Isolate Valency (200ms)
Mg2+
before
→
Mg2
after
± signs fade to opacity:0. Bare integer remains, repositioned as plain superscript. Same for anion.
Framer: animate={{ opacity: 0 }} on ± span. Integer re-renders without ± wrapper.
Frame ② — Criss-Cross Travel (600ms)
Mg2
Cl1
Each integer is a motion.span with layoutId. Animates via x/y offsets across the bridge. Uses transition: { type:'spring', stiffness:120 }.
Frame ③ — Polyatomic Bracket Check (300ms, conditional)
Mg
⇌
(OH)2
Only fires if: target is polyatomic AND incoming subscript > 1.
Brackets rendered as two motion.spans that drop from y:-20, opacity:0 → resting position.
Skip entirely if subscript === 1 (e.g. NaOH — no brackets needed).
Frame ④ — GCD Simplification (400ms, conditional)
Ca2O2
raw
÷ 2
CaO
simplified
GCD(cation_subscript, anion_subscript) computed in JS. If GCD > 1:
1. Show "÷ N" badge fading in above formula (500ms)
2. Subscript values animate to simplified numbers (or disappear if === 1)
3. ÷ badge fades out
Skip entirely if GCD === 1 (e.g. MgCl₂ — no simplification needed).
